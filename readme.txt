== Description ==

Chat Fox is a 100% FREE live chat application for your website. Get all the premium features others are paying for, absolutely free!

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins` directory, or install the plugin through the *Plugins* screen.
1. Activate the plugin through the *Plugins* screen.
1. Sign up for a **FREE** account on Chat Fox (<a href="http://chatfox.co/" target="_blank">http://chatfox.co/</a>) then **VERIFY** and **CONFIRM** your email address.
1. <a href="https://live.chatfox.co/operator/index.php" target="_blank">Log in</a> to your account and add your domain(s) to the white list (under "Chat Widget" on your dashboard).
1. Activate the chat widget by enabling it under the *WP Chat Fox* -> *Settings* screen.
